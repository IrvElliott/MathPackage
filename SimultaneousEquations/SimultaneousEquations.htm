SOLVING SIMULTANEOUS EQUATIONS

 This chapter presents some ways to solve a set of simultaneous equations. The problem consists of n linear equations for n variables, represented by n equations, each with the sum of multiples of n independent variables on the left and a resultant value on the right.  The goal is to determine the values of the n variables.

 The method of solution is based on a process called "Gaussian Elimination", and is based upon two facts:

(1)   You can multiply any row of the problem by a constant without changing the solution.

(2)    You can add or subtract one row from another without changing the solution.

  An example for n=4 follows:                                    

 4*x1 + 6*x2 + 3*5x3 + 9*x4 = 72

 2*x1 + 5*x2 + 7*5x3 + 8*x4 = 98                                                                

6*x1 + 8*x2 + 3*9x3 + 4*x4 = 56                                                     

2*x1 + 5*x2 + 6*5x3 + 7*x4 = 62

 The problem can be represented as an n by n matrix X and a vertical n-element vector Y, as follows:

  4  6  3  9    72

 2  5  7  8     98

 6  8  3  4     56

  2  5  6  7     62

The first and easiest method is to select the diagonal set of numbers starting at the upper left corner of X  as  “pivot points”.  In the example, these are 4, 5, 3 and 7.  Now, do the  following for each pivot:

(1)   divide each value in the pivot row, including both X and Y values, by the pivot point value. This will make the pivot point equal to one.

(2)    For each non-pivot row,

 (a) Take the value that is both in that row and is also in the same column as the pivot and call

       that value v.

 (b)  For each non-pivot row, subtract v times each element of the pivot row from each

        corresponding element of the non-pivot row.

 Doing this for the example would give:

     1  0  0  0      71.0000

     0  1  0  0     -60.6667

     0  0  1  0       28.6667

     0  0  0  1         7.3333

    This leaves the solution in the vector on the right:

     x1 =  71   x2=-60.6667   x3=28.6667   x4=7.3333

 If it is impossible to reduce the  X array to an array of 1's in the diagonal and otherwise 0's, the original problem is invalid and thus has no solution.

 A second method is said to reduce error in the results. It is like the first method, but the pivots are not necessarily on the diagonal and the rows are not necessarily processed from top to bottom.  Thus, the resultant vector is scrambled and must be unscrambled after the n executions.  Two more vectors are added to the problem. These are a vertical n-element row vector R  at the left of X and a horizontal column vector C above X. These are used to record the selections of pivots so that the resulting answers can be unscrambled.. Initially, R and C are set to zero; during processing, each pivot is recorded by placing its column number into the R vector element to the left and a 1 into the C vector element above.The procedure consists of  n executions of the following steps:

      (1)   Set each value in vectors R and C  to zero.

(2)   Perform the following steps n times:

          (a) Examine all of the matrix X values that  are not in a row marked  in R nor a column marked

                in C. Choose the value that is largest in absolute value as the pivot, then record it in R and C.

           (b) Divide each element  in the pivot's row by the pivot value. For each row not belonging to the

                 pivot's row, take the value that is in that row and is also in the same column as the pivot and

                 call that value v.

           (c) Subtract v times each element of the pivot row from each corresponding element of all other

                 rows that are unmarked in R.

       (3) Unscramble the Y vector results, using the column values in the R vector as  a guide.

A third method is to use determinants. The method is described in the DETERMINANTS section.
