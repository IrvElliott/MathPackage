VECTORS AND MATRICES

In this document, multiplication is indicated by a star (*) and division is indicated by a slash (/). This is done because the x-style multiplication sign and dash with two dots for divide are not available in most computer character fonts.

A "vector" is like an arrow flying in a certain direction, with a head, a tail, and a magnitude (length). In a two-dimensional world, such as  a map, a vector can be represented on paper as an arrow. The arrow length can represent some variable such as length or velocity. A series of vectors can thus represent a path taken on the ground, or the directions and velocity of a traveling vehicle. Mathematically, a vector is usually represented by a name topped with a short horizontal bar that is sometimes followed by a set of n value "elements" in square brackets. The elements can be either numeric values or variable expressions. The "dimension" of the vector is n, the number of elements. In the text of this document, the bar over the vector name is skipped because it is not printable in the letter font. Some examples for a three-dimensional vector are as follows:

 A[6,7 9]    B[ i, j, k]    C[3, k+m, 17] 

Each element denotes a displacement in an n-dimensional reference space: for example, for two dimensions, the horizontal and vertical directions on a graph or the east and north directions on a map. The vector notation does not necessarily specify the location of a vector in the reference space. It just specifies the x and y differences from the head to the tail.

VECTOR SCALAR OR DOT PRODUCT 

Another procedure that produces a numerical value is the "scalar product" but is often called the "dot product" because it is represented  by placing a dot between two vector names. Case 2 of the previous illustration shows two vectors A and B with an angle theta between them. The dot product is formed by multiplying each element of A by the corresponding element of B and adding the results together. The result  is equal to the magnitude of A multiplied by the magnitude of B multiplied by the cosine of the angle between A and B. This is useful when something is being designed and there is a need to know the angle between two straight pieces. One can form the dot product, divide it by both the magnitude of A and B, then look up the angle in a table of cosines.

THREE-DIMENSIONAL VECTORS

We live in a three-dimensional world. Three-dimensional vectors are useful for many tasks, such as guiding aircraft, creating 3-D displays and model objects for 3-D printers. Three-dimensional vectors can be added or subtracted and the magnitude can be calculated by using the same methods used for the two-dimensional ones, except that the value of "n" is 3. Also, in 3-D there is another mathematical operation called the "cross product". The input for that operation is two vectors. Any two vectors A and B not pointing in the same direction establish a flat "plane" between them. The operation creates a third vector C that is perpendicular to that plane. The direction of C is away from one side of the plane or the other, depending upon whether the operation is performed on A followed by B, or on B followed by A. The magnitude of the third vector is |A|*|B|*sin(theta), where theta is the angle between the vectors. The following diagram can be used as an aid to remembering how to calculate the cross product.

The cross product is useful in designing, displaying and 3-D printing three-dimensional objects. The skin of the object is represented as covered with small triangles. Each triangle is represented by three points that provide data for vectors. If the points are specified in a counter-clockwise order, the cross product vector for the triangle points to the inside of the object; otherwise, to the outside, During a 3-D display, the triangles having cross-product vectors pointing away from the viewer are on the far side of the object and thus are not visible.

N-DIMENSIONAL VECTORS

A column or row of n numbers can be treated as a vector. For example, solving a set of simultaneous equations for 10 variables involves multiplying, adding and subtracting what amounts to 10-dimensional vectors. For that application, not only rows of values but also columns of values are treated as vectors. Many operations on matrices involved steps involving rows and columns treated as vectors.

The vector add, subtract, magnitude and dot product operations for vectors of more than dimension 3 are similar to those for dimensions 2 and 3-- just more terms to handle. One difference is that the "cosine of the angle between" is called the "correlation coefficient". It is equal to 1 if the two vectors are identical or multiples of each other; if they are not and the coefficient is 0, the two vectors are said to be "orthogonal" to each other, which is hard to imagine but is of great use when performing experiments.

MATRICES

A matrix is a rectangular array of values with m horizontal rows and n vertical columns, where m and n are integers. Matrices are useful for performing many calculations such as the solving of simultaneous equations, plotting contours, creating data for 3-D displays and 3-D printing, forecasting the weather and simulating aircraft flights. The symbol for a matrix consists of a name. Sometimes, the name is surrounded by square brackets, as follows: [A]. When accessing an element of a matrix, the matrix name is used followed by two integer indices in square brackets, as follows: A[2,5]. The indices refer to the row number (top row = 1) and the column number (left column = 1) of the element.

SQUARE MATRIX OPERATIONS

Square matrices are used in many mathematical procedures. One operation that is a part of most of these procedures is to reduce the matrix to a unit matrix by performing the following row operations: multiply all elements in a row by a constant, add or subtract elements of a row from those of another row or interchange rows. One example of a reduction procedure is as follows:

For each row i,

  Divide all elements in the row by the "pivot" element A(i,i). This leaves the pivot at 1.

  For each other row j,

      Determine a multiplier V from element A(j,i) that is in the same column as the pivot.

      Subtract V times each element of the pivot row from the corresponding element of row j. This zeroes out the element A(j,i).

  End of row j operations.

End of row i operations.

SIMULTANEOUS EQUATIONS

If one is provided by n equations in n independent variables, there is a convenient matrix method of solving them. For example, for three variables A B and C, one would set up a matrix and an n-vector as follows:

6*A + 5*B + 12*C = 42                     6   5   12    42

4*A + 3*B + 10*C = 30          ----->  4   3   10   30  

5*A + 6*B + 11*C = 56                      5   6   11   56

As shown in the third area of the diagram, the square of  left-side values X is appended to the right by the vector Y of right-side values. To solve the system, n row operations are performed on the combined n by n+1 matrix to reduce the X part to a unit I matrix. The resulting values of the Y vector will then be the answers for variables A, B and C.

CORRELATION

When some square matrix operations are performed on a computer, a problem may occur when one row is a multiple or equal of another. For example, for a set of simultaneous equations, one such row can be multiplied then subtracted from the other and the system would then have a zero row and the problem would be unsolvable. A similar problem may occur if a row or column is almost a multiple of another. This may causes a numeric overflow an attempt to form a number that is too large to represent in the computer's system. An attempt to find the inverse of such a matrix will fail. Such a matrix is called "singular", which is another way of saying it is bad.

To check for these conditions, we can calculate the "correlation coefficient" for each combination of two columns. As was explained in the VECTOR discussions of this document, the dot product of two vectors of dimension two or three produces a result of |A|*|B|*cos(theta), where theta is the angle between them. When the dot product of two vectors of dimension greater than three is formed, the result is considered to be |A|*|B|*cc, where cc is the correlation coefficient. Like the cosine, the value ranges from 0 to 1. If the value is 1, the two vectors are multiples or equals of each other; if the value is 0, they are said to be "orthogonal".

VECTOR  ROTATION

The rules for rotating a vector by an angle theta assume that the X-Y-Z coordinate system is in accordance with the "right-hand rule". The cross-product of the X axis and the Y axis should result in a Z axis vector. In other words, if a person holds up a right hand, points the thumb up and curves the fingers as to move the X axis to the Y,  the thumb will point up to represent Z. For example, X could point east, Y then would point south and Z would point upward.
